Essa é a segunda versão do código do projeto, foram implementadas a ligação com o banco de dados. Favor, criar um banco de dados com uma database correspondente, e fazer a devida conexão para que o código funcione.

Problemas encontrados:

- Banco de dados incompleto, necessita de Querys para consulta.
- Interface gráfica incompleta, falta a interface de consulta dos comprovantes.
- Não há funções nos botões nem nos campos de texto da interface gráfica.
- Por hora, a entrada de dados nas tabelas não enviam informações para o banco de dados.
